When the user writes **mycopyright**, replace it with: **Copyright © 2026 MMFFDev. All rights reserved.**

**Why:** User shorthand for their copyright notice — avoids retyping it every time.
**How to apply:** Any time "mycopyright" appears in a user message, treat it as the full copyright string. Use it in code, files, comments, or wherever the user places it.
