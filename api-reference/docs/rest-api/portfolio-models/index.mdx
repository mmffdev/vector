## POST /api/portfolio-models/:id/adopt

<EndpointHeader method="POST" path="/api/portfolio-models/:id/adopt" auth="bearer" />
<AuthGate role="padmin" />
<TableDeps tables={['subscription_portfolio_model_state', 'subscription_layers', 'subscription_workflows', 'subscription_workflow_transitions', 'subscription_artifacts', 'subscription_terminology']} />

Starts the adoption saga for the model. Returns `202 Accepted` immediately. Stream progress via `/adopt/stream`.
Chi route pattern: `/api/portfolio-models/{id}/adopt`

The saga mirrors 6 step groups from `mmff_library` into `mmff_vector`:
1. Layers
2. Workflows
3. Workflow transitions
4. Artifacts
5. Terminology
6. State record

**Errors**

| Status | JSON `code` | Meaning |
|---|---|---|
| `409` | `ALREADY_ADOPTED` | This model is already the active adoption |
| `409` | `ADOPT_IN_PROGRESS` | Another adoption is in progress |

**Response** — `202 Accepted`

```json
{ "streamPath": "/api/portfolio-models/<id>/adopt/stream" }
```

Handler: `internal/portfoliomodels/adopt.go`

## GET /api/portfolio-models/:id/adopt/stream

<EndpointHeader method="GET" path="/api/portfolio-models/:id/adopt/stream" auth="bearer" />
<AuthGate role="padmin" />
<SseBadge />

SSE stream of adoption saga progress. Chi route pattern: `/api/portfolio-models/{id}/adopt/stream`

**Must use `fetch()` — not `EventSource`** (EventSource cannot set `Authorization` header).

```ts
const res = await fetch(streamPath, {
  headers: { Authorization: `Bearer ${token}` },
});
const reader = res.body!.getReader();
```

**Event types**

| Event | Data shape | Meaning |
|---|---|---|
| `step` | `{"step":"layers","ok":true,"msg":"Applied 4 layers"}` | One saga step completed |
| `done` | `{"success":true}` | Adoption fully complete |
| `error` | `{"message":"..."}` | Adoption failed |

On an idempotent call (already adopted), the stream emits 7 synthetic `step` events followed by `done` so the overlay completes normally.

Handler: `internal/portfoliomodels/adopt_stream.go:111`
