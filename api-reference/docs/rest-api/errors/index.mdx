## POST /api/errors/report

<EndpointHeader method="POST" path="/api/errors/report" auth="bearer" />
<AuthGate role="any" />
<TableDeps tables={['error_events']} />

Reports a client-side error event. The error code is validated against the `mmff_library.error_codes` catalogue before insertion into `mmff_vector.error_events`.

**Request body**

```json
{
  "code": "string",
  "message": "string",
  "context": {}
}
```

**Response** — `204 No Content`

**Errors**

| Status | Meaning |
|---|---|
| `400` | Unknown error code (not in catalogue) |

Handler: `internal/errorsreport/handler.go:78`
