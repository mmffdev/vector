# Admin

Base path: `/api/admin` — `internal/users/handler.go`, `internal/permissions/handler.go`

All routes require bearer auth + fresh password.
Chi pattern for parameterised routes: `/api/admin/users/{id}`, `/api/admin/permissions/{id}`

## Users

### GET /api/admin/users

<EndpointHeader method="GET" path="/api/admin/users" auth="bearer" />
<AuthGate role="gadmin" /> <AuthGate role="padmin" />
<TableDeps tables={['users']} />

Lists all users in the subscription.

**Response** — `200 OK`

```json
[
  {
    "id": "string",
    "email": "string",
    "role": "gadmin | padmin | user",
    "createdAt": "string",
    "archivedAt": "string | null"
  }
]
```

Handler: `internal/users/handler.go:58`

---

### POST /api/admin/users

<EndpointHeader method="POST" path="/api/admin/users" auth="bearer" />
<AuthGate role="gadmin" />

Creates a new user in the subscription. Sends an invite email.

**Request body**

```json
{
  "email": "string",
  "role": "gadmin | padmin | user",
  "sendInvite": true
}
```

**Response** — `201 Created`

```json
{ "id": "string" }
```

**Errors**

| Status | Meaning |
|---|---|
| `409` | Email already exists in this subscription |

Handler: `internal/users/handler.go:30`

---

### PATCH /api/admin/users/:id

<EndpointHeader method="PATCH" path="/api/admin/users/:id" auth="bearer" />
<AuthGate role="gadmin" />

Updates a user's role or archived state. Partial update — omit fields to leave unchanged.
Chi pattern: `/api/admin/users/{id}`

**Request body**

```json
{ "role": "gadmin | padmin | user", "archivedAt": "string | null" }
```

**Response** — `204 No Content`

Handler: `internal/users/handler.go:73`
